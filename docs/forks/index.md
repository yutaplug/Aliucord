# Plugin forks

To install them, download the plugin .zip and move it to the `Aliucord/plugins` folder using a file manager & restart Aliucord in case it was open.

- [AudioPlayer](https://github.com/Archimedes9500/Halkiion-rushiiMachine-aliucord-plugins/raw/builds/AudioPlayer.zip) fork by Halkion fixes crashes and some bugs with audio files.

- [Waifuim](https://github.com/OasisVee/AliucordPlugins3/raw/builds/Waifuim.zip), [NekosLife](https://github.com/OasisVee/AliucordPlugins3/raw/builds/NekosLife.zip) forks by Serinova fix the commands being fully broken. Additionally, Serinova made a plugin called [AnimeImageFetch](https://github.com/OasisVee/my-plugins/raw/builds/AnimeImageFetch.zip) containing NSFW since NekosLife doesnt contain NSFW anymore.

- [CheckLinks](https://github.com/OasisVee/AliucordPlugins2/raw/builds/CheckLinks.zip) fork by Serinova fixes the majority of the urls not being checked (VirusTotal changed its link structure).

- [NitroSpoof](https://github.com/kiwi-706/AliucordPlugins/raw/builds/NitroSpoof.zip) fork by Kiwi makes the fake emojis look real just like on Vencord's FakeNitro plugin or Vendetta's (and their forks) Realmoji plugin. Also fixes a scrolling bug in emoji picker.

- [FakeStickers](https://github.com/Archimedes9500/aliucord-plugins/raw/builds/BetterFakeStickers.zip) fork by Archimedes (renamed to BetterFakeStickers) fixes not being able to reply with stickers.

- [Ip](https://github.com/OasisVee/AliucordPluginsSc/raw/builds/Ip.zip) fork by Serinova fixes the `/ip` command.

- [SendEmbeds](https://github.com/OasisVee/aliucord-pluginsC/raw/builds/SendEmbeds.zip) fork by Serinova makes the `/embed` command work again by using directwebhook (original API died).

- [PersistSettings](https://github.com/Archimedes9500/plugins/raw/builds/PersistSettings.zip) fork by Archimedes makes the plugin work again in 126.21 version.

- [PlayableEmbeds](https://github.com/Enovale/AliucordPlugins/raw/builds/PlayableEmbeds.zip) fork by Enovale makes the embeds work for every site. Don't use this if you want working YouTube embeds.

- [PlayableEmbeds](https://github.com/yutaplug/Aliucord/raw/main/stuff/PlayableEmbeds.zip) fork by orange32 fixes YouTube embeds.
