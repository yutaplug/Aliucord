# How to get your Discord account token on mobile
**Note:** the extension provided is open source, you can manually check the [source code](https://github.com/SabbirXavier/Discord-Token-Extractor-chrome-extension) by yourself if you don't trust it.

- Install [Cromite browser](https://github.com/uazo/cromite/releases/latest/download/arm64_ChromePublic.apk)
- Open it & go to Developer options in settings
- Enable "Enable extensions" option
- Download [this extension](https://github.com/SabbirXavier/Discord-Token-Extractor-chrome-extension/releases/latest/download/Discord.Token.Extractor.zip)
- Extract the .zip with a file manager
- In Cromite, click the three dots (â‹®) and go to "Extensions", inside it, enable "Developer mode"
- Press "Load unpacked" and select the extension folder (the one that contains all the files, not the empty one)
- Login to [Discord web](https://discord.com/login)
- Click the ðŸ§© icon and open the extension, now click "Extract Token"
