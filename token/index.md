# How to get your Discord account token on mobile

- Install [Cromite browser](https://github.com/uazo/cromite/releases/latest/download/arm64_ChromePublic.apk)
- Open it & go to Developer options in settings
- Enable "Enable extensions" option
- Download [this extension](https://github.com/SabbirXavier/Discord-Token-Extractor-chrome-extension/releases/download/latest/Discord.Token.Extractor.zip)
- Extract the .zip with a file manager
- Press 3 dots in cromite, go to "Extensions" and inside it enable "Developer mode"
- Press "Load unpacked" and select the extracted folder
- Login to Discord web
- Click the extension
